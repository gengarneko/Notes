## 盛赞 Redis

- 高性能 Key-Value 服务器（QTS达到十万级别）
- 多种数据结构（五大数据结构以及衍生结构和算法）
- 丰富的功能（查询，发布订阅，QEO）
- 高可用分布式支持



## 适合人群

- Redis 初学者
- Redis 进阶者
- 希望了解 Redis 企业级开发运维技巧



## 技术储备

- 了解 Linux 命令的基本使用
- 了解常用数据结构
- 了解一门编程语言



## 授课思路

- 开发运维结合
- 原理接入
- 实战案例分析



## 课程目标

- 全面了解 Redis 单机的相关功能
- 全面了解 Redis 高可用和分布式方案
- 理解企业级 Redis 的开发运维
