3.4--布局中的链接

布局中的链接

视图中的一些代码仍然可以进行优化,比如链接地址可以改成

<li><a href="{{ route('help') }}">帮助</a><li>

{{ }} 是 HTML 中内嵌 PHP 的 Blade 语法标识符 表示包含该区块内的代码都将使用 PHP 来编译运行
route() 方法由 Laravel 提供 通过传递一个具体的路径名来生成完整的 URL 

Laravel 路由

如果需要用下面的方式来渲染 help 链接 ,需要先为路由定义 help 路由名称

<li><a href="{{ route('help') }}">帮助</a></li>

Laravel中,可以通过在路由后面链式调用 name 的方法为路由指定名称

Route::get('/help', 'StaticPagesController@help')->name('help');

当我们在页面渲染路由时, 

http://sample.test/help

route('help') 为我们生成了完整的 URL 地址, 这样当我们需要对生成的 URL 进行更改时,我们只需要改路由文件即可
由此可见在实际开发中对路由命名是一个好习惯

假如我们的应用中由 10 个地方使用 route('help') 方式书写 URL 地址,因为特殊原因,需要将 help 改成 fuck 这时候我们只需要改动路由文件即可
