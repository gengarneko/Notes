2. 使用 JS 创建 HTML
3. 添加 CSS
4. 添加事件监听

代码和预览：<https://github.com/FrankFang/nav-demo>

注意：
代码中的 `return JSON.parse(localStorage.getItem(name) || '')`
要改成 `return JSON.parse(localStorage.getItem(name) || 'null')`
