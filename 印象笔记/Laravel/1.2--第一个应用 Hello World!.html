1.2--第一个应用 Hello World!

先开启homestead 虚拟机
    cd ~/Homestead && vagrant up
    vagrant up

开启成功后,新建一个名为 Laravel 的项目
    cd ~/Code
    composer create-project laravel/laravel Laravel --prefer-dist  "5.5.*"

进入 homestead.test 看到以下页面



composer 

composer 是一款跨平台的 PHP 依赖管理工具,创作灵感来源于 Node.js 的 NPM 和 Ruby 的 Bundler
Laravel 使用 Composer 来作为扩展包的管理工具
可以利用 Composer 结合其他开源扩展包来达到快速建站的目的
打开 composer.json 可以看到 Laravel 默认集成了一些支持框架的扩展包

Composer 是一个伟大的发明,他让组件编程成为可能,编写程序像拼接乐高,极大地提高了开发的效率和代码的可复用性,解放了生产力
