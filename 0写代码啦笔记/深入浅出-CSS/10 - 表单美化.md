## 百度搜索栏

```css
body {
  border: 1px solid red;
  min-height: 100vh;
}
html {
  height: 100%;
}
```

我们使用 vh 来使得我们的容器高度为浏览器视图高度，以完成居中操作。

> 第一个工具：caniuse.com 告诉你能不能用一些属性
