1.2 -- 创建应用及基础配置

Composer 加速

在创建项目之前, 首先实现 Composer 的加速安装

$ composer config -g repo.packagist composer https://packagist.phpcomposer.com

创建 LaraBBS 应用

下面使用 Composer 创建一个名为 LaraBBS 的应用

$ cd ~/Code
$ composer create-project laravel/laravel RepairSystem --prefer-dist "5.5.*"

配置信息存放位置

laravel 所有的配置信息都存放在 config 目录中, 每个文件中包含多个配置选项


访问配置值

使用全局 config 函数来访问配置值. 配置值的访问可以使用点语法, 包含了要访问的 文件名称 和 选项名称. 还可以指定默认值

$value = config('app.timezone');

要在运行时设置配置值, 传递一个数组给 config 函数

config(['app.timezone' => 'America/Chicago']);
