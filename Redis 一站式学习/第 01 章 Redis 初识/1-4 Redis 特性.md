## Redis 的特性

1. 速度快（内存数据库特性）
2. 持久化
3. 多种数据结构
4. 支持多种编辑语言
5. 功能丰富
6. 简单
7. 主从复制
8. 高可用，分布式



## 特性1 - 速度快

官方宣称能达到 10W OPS

数据存在哪？ ———— 内存

什么语言写？ ———— C 语言（5w line）

线程模型 ———— 单线程

## 特性2 - 持久化

Redis 将所有数据保存在内存中，对数据的更新将异步地保存到磁盘上。



## 特性3 - 多种数据结构

![1542095861037](E:\笔记\Redis 一站式学习\第 01 章 Redis 初识\1-4 Redis 特性.assets\1542095861037.png)

其他的数据结构：

- BitMap：位图（小内存实现高效存储，比如过滤器）	本质是字符串
- HyperLogLog：超小内存惟一值计数，12K（可以达到上十亿），技术有误差      本质是字符串
- GEO：地理信息位置
