# 1 - 初始化

我们使用 `vue-cli 3.0` 进行项目搭建：

```
winpty vue.cmd create hello-world
```

我们将建好的仓库与之合并，然后选择手动添加组件：

![](https://i.loli.net/2019/03/30/5c9ed59fcaa44.png)

然后我们上传文件：

```
git add .
git commit -m "create project"
git push
```
