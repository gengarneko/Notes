## Redis 常用配置

| 配置项    | 说明                          | 默认值 |
| --------- | ----------------------------- | ------ |
| daemonize | 是否是守护进程（no\|**yes**） | no     |
| port      | Redis 对外端口号              | 6379   |
| logfile   | Redis 系统日志                |        |
| dir       | Redis 工作目录                |        |

为什么是6379？  因为是手机按键上 MERZ（意大利女歌手 Alessia Merz） 对应的号码

还有配置项：

- RDB config
- AOF config
- slow Log config
- maxMemory...
