## 伪代码的好处

1. 不用纠结于语法的细节，因为语法是你自己定的
2. 可以体会设计者的想法，因为语法是你自己定的

我的语法：

- a <- 1 表示将 1 拷贝给 a

- 条件控制语句

  ```
    if xxx
        1
    elseif yyy
        2
    else
        3
    end
  ```

- 循环语句

  ```
    n <- 0
    while n < 10
        print n
        n <- n+1
    end
  ```

- length 表示一个容器, 'length' 表示字符串, length <- 'length' 表示将字符串放入容器



## 流程图

### 一个最简单的例子

从正整数组 a 中找出最小的数字，打印出来

```
a <- {
    '0': 23
    '1': 43
    '2': 239
    '3': 1321
    '4': 90
    'length': 5
}
min <-  a['0']
index <- 1
while index < a['length']
    if a[index] < min
        min <- a[index]
    end
    index <- index + 1
end
print min
```
